# frontend Shop

## Build Setup
Архитектура поделена на два nuxt-проекта

###Influencer
```bash
# install dependencies
$ npm install

#move in directory
$ cd influencer
# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
